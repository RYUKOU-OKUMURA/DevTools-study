## DevTools って何？

**Chrome DevTools（デベロッパーツール）** は、Google Chrome に標準で搭載されている開発者向けツールです。

### こんな時に使います

- **エラーが出た時**：どこでエラーが起きているか確認できる
- **デザインを調整したい時**：色やサイズをその場で試せる
- **要素を探したい時**：ボタンや画像のコードがどこにあるか見つけられる
- **ページの表示が遅い時**：何が遅くしているのか調べられる

プログラミング初心者でも、DevTools を使えるようになると開発がグッと楽になります！

---

## DevTools を開く方法

### 方法1: キーボードショートカット（一番速い！）

**Windows / Linux:**
```
Ctrl + Shift + I
```

**Mac:**
```
Command + Option + I
```

### 方法2: 右クリックメニュー

1. ページ上の任意の場所を **右クリック**
2. **「検証」** または **「要素を検証」** を選択

この方法は、特定の要素を調べたい時に便利です！

### 方法3: メニューから開く

1. Chrome の右上の **︙**（3点メニュー）をクリック
2. **「その他のツール」** → **「デベロッパーツール」** を選択

## 画面の見方

DevTools を開くと、画面が大きく2つに分かれます。

```
┌─────────────────────────────────────┐
│     実際のウェブページ              │
│                                     │
├─────────────────────────────────────┤
│     DevTools パネル                 │
│  [Elements][Console][Network]...    │
│                                     │
└─────────────────────────────────────┘
```

### 表示位置を変える

DevTools の右上にある **︙**（3点メニュー）から、表示位置を変更できます：

- **下部に表示**（デフォルト）
- **右側に表示**
- **左側に表示**
- **別ウィンドウで開く**

画面サイズに合わせて、見やすい位置に変更しましょう！

## よく使うパネルの概要

DevTools には様々なパネル（タブ）がありますが、初心者がまず覚えるべきは以下の3つです：

### 1. **Elements パネル**
HTML と CSS を確認・編集できます。

**こんな時に使う：**
- 「このボタンの色を変えたい」
- 「この文字のサイズを変えたい」
- 「この要素がどこにあるか探したい」

→ 詳しくは [Elements パネル編](./02_elements.md) へ

### 2. **Console パネル**
JavaScript のエラーやログを確認できます。

**こんな時に使う：**
- 「ページにエラーが出ている」
- 「JavaScript が動いていない」
- 「コードの動作を確認したい」

→ 詳しくは [Console パネル編](./03_console.md) へ

### 3. **Network パネル**
ページが読み込むファイルや通信を確認できます。

**こんな時に使う：**
- 「画像が表示されない」
- 「API の通信がうまくいかない」
- 「ページの読み込みが遅い」

→ 詳しくは [Network パネル編](./04_network.md) へ

---

## パネルの切り替え方

上部のタブをクリックするだけで、パネルを切り替えられます。

```
[Elements] [Console] [Sources] [Network] [Performance] ...
   ↑         ↑         ↑         ↑          ↑
  クリック  クリック  クリック  クリック   クリック
```

タブが隠れている場合は、右側の **»** をクリックすると表示されます。

## まとめ

- DevTools は Chrome に標準搭載されている開発者向けツール
- `Ctrl + Shift + I`（Mac: `Command + Option + I`）で開く
- まずは **Elements**、**Console**、**Network** の3つを覚えよう
- 画面の表示位置は自由に変えられる
