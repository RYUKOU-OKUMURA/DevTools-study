## Elements パネルとは

**Elements パネル** は、ページの **HTML（構造）** と **CSS（見た目）** を確認・編集できるパネルです。

### こんなことができます

- ボタンや画像などの要素がどこにあるか探せる
- 文字の色やサイズを変えて試せる
- 要素の配置を調整できる
- CSS がどこで定義されているか確認できる

**重要：** ここでの変更は **一時的** です。ページをリロード（更新）すると元に戻ります。

## 要素を選択する方法

### 方法1: 検証ツールを使う（一番簡単！）

1. DevTools の左上にある **矢印アイコン**（要素選択ツール）をクリック
   - ショートカット：`Ctrl + Shift + C`（Mac: `Command + Option + C`）

2. ページ上のカーソルを動かすと、要素がハイライトされます

3. 調べたい要素をクリック

→ Elements パネルにその要素の HTML が表示されます！

```
┌─────────────────────────────────────┐
│  ページ                             │
│    ┌────────┐  ← この部分を          │
│    │ボタン  │     クリック            │
│    └────────┘                       │
├─────────────────────────────────────┤
│  Elements                           │
│  <button class="btn">ボタン</button>│
│      ↑ 該当するコードが表示される    │
└─────────────────────────────────────┘
```

### 方法2: 右クリックから検証

1. ページ上の調べたい要素を **右クリック**
2. **「検証」** を選択

→ すぐにその要素が選択された状態で DevTools が開きます！

### 方法3: HTML から探す

Elements パネルで HTML のツリーを展開して探すこともできます。

- **▶** をクリックして要素を展開
- **▼** で要素が展開されている状態

## HTML を確認・編集する

### HTML の構造を見る

Elements パネルでは、HTML がツリー構造で表示されます。

```html
<body>
  <div id="app">
    <h1>タイトル</h1>
    <button class="btn">クリック</button>
  </div>
</body>
```

- **親要素** と **子要素** の関係がわかりやすく表示される
- 要素にマウスを乗せると、ページ上でハイライトされる

### HTML を編集してみる

#### テキストを変更する

1. 変更したい要素をダブルクリック
2. テキストを編集
3. `Enter` キーを押す

→ ページ上の表示が即座に変わります！

#### 属性を編集する

要素を右クリックして以下の操作ができます：

- **「属性を追加」**：新しい属性を追加
- **「属性を編集」**：既存の属性を変更
- **「HTML として編集」**：HTML 全体を編集

#### 要素を削除する

1. 削除したい要素を選択
2. `Delete` キーを押す

または右クリックして **「要素を削除」** を選択

## CSS を確認・編集する

### CSS を見る場所

Elements パネルで要素を選択すると、右側（または下部）に **Styles（スタイル）** パネルが表示されます。

```
┌─────────────────┬─────────────────┐
│   HTML ツリー   │   Styles        │
│                 │  .btn {         │
│   <button       │    color: blue; │
│     class="btn">│    padding: 10px│
│   </button>     │  }              │
└─────────────────┴─────────────────┘
```

### CSS の見方

**Styles パネルには以下が表示されます：**

1. **element.style**：インラインスタイル（HTML に直接書かれたスタイル）
2. **セレクタ名**：CSS ファイルで定義されたスタイル
3. **取り消し線**：上書きされて無効になっているスタイル
4. **ファイル名とリンク**：そのスタイルが定義されているファイル（クリックでジャンプ）

### CSS を編集してみる

#### 色を変更する

1. 色のコードの横にある **カラーパレット** をクリック
2. 好きな色を選ぶ

→ リアルタイムで色が変わります！

#### 値を変更する

1. 変更したいプロパティの値をクリック
2. 新しい値を入力
3. `Enter` キーで確定

**例：**
```css
padding: 10px;  ← ここをクリックして 20px に変更
```

#### 新しいプロパティを追加する

1. スタイルブロックの中をクリック
2. プロパティ名を入力（例：`color`）
3. `Tab` キーで値の入力に移動
4. 値を入力（例：`red`）

#### プロパティを無効にする

チェックボックスをクリックすると、そのプロパティを一時的に無効化できます。

```css
☑ color: blue;   ← チェックを外すと無効化
☐ color: blue;   ← 無効化された状態
```

## 実践：よくある使い方

### ケース1：ボタンの色を変えたい

1. **検証ツール**（矢印アイコン）をクリック
2. ボタンをクリックして選択
3. 右側の **Styles** パネルで `background-color` を探す
4. カラーパレットで好きな色を選ぶ
5. 気に入ったら、その CSS コードを実際のファイルにコピー

### ケース2：文字サイズを調整したい

1. テキストを右クリックして **「検証」**
2. Styles パネルで `font-size` を探す
3. 値を変更して試す（例：`16px` → `20px`）
4. 最適なサイズが決まったら、実際のファイルに反映

### ケース3：要素が表示されない原因を探す

1. 表示されるはずの要素を選択
2. Styles パネルで以下をチェック：
   - `display: none;` になっていないか
   - `visibility: hidden;` になっていないか
   - `opacity: 0;` になっていないか
   - 親要素で `overflow: hidden;` になっていないか

### ケース4：レイアウトが崩れている原因を探す

1. 崩れている要素を選択
2. **Computed（計算済み）** タブを開く
3. ボックスモデル図を確認：
   ```
   ┌─────────────────────┐
   │  margin             │
   │  ┌───────────────┐  │
   │  │ border        │  │
   │  │ ┌──────────┐  │  │
   │  │ │ padding  │  │  │
   │  │ │ ┌──────┐ │  │  │
   │  │ │ │コンテンツ│ │  │
   │  │ │ └──────┘ │  │  │
   │  │ └──────────┘  │  │
   │  └───────────────┘  │
   └─────────────────────┘
   ```
4. 予想外の `margin` や `padding` がないか確認

## 便利な機能

### 1. ホバー状態を確認する

ボタンなどにマウスを乗せた時のスタイル（`:hover`）を確認したい時：

1. 要素を選択
2. Styles パネルの右上にある **:hov** をクリック
3. **:hover** にチェックを入れる

→ マウスを乗せなくてもホバー状態を確認できます！

### 2. 要素のクラスを追加・削除する

Styles パネルの上部にある **.cls** をクリックすると、クラスの追加・削除ができます。

### 3. Box Model を視覚的に確認する

要素を選択すると、ページ上に以下が色分けされて表示されます：

- **青色**：コンテンツ領域
- **緑色**：パディング
- **黄色**：ボーダー
- **オレンジ色**：マージン

### 4. スクリーンショットを撮る

要素を選択 → 右クリック → **「スクリーンショットをキャプチャ」**

- **「ノードのスクリーンショットをキャプチャ」**：選択した要素のみ
- **「フルサイズのスクリーンショットをキャプチャ」**：ページ全体

### 5. CSS の定義元にジャンプする

Styles パネルのファイル名（例：`style.css:15`）をクリックすると、そのファイルの該当行にジャンプできます。

---

## まとめ

- Elements パネルで HTML と CSS を確認・編集できる
- **検証ツール**（矢印アイコン）を使って要素を選択するのが簡単
- Styles パネルで CSS をリアルタイムで編集できる
- 変更は一時的なので、気に入ったら実際のファイルに反映する
- `:hover` などの擬似クラスも確認できる
