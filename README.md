# Chrome DevTools 完全ガイド

**非エンジニアのプログラミング初学者向け**

個人開発で必ず役立つ、Chrome DevTools の使い方を基礎から学べる教材です。

---

## この教材について

この教材は、プログラミングを始めたばかりの人が、Chrome DevTools を使って以下のことができるようになるために作られました：

- **エラーログを確認する**：どこでエラーが起きているか見つけられる
- **エレメントを探す**：ボタンや画像のコードがどこにあるか特定できる
- **デザインを調整する**：色やサイズをその場で試せる
- **通信を確認する**：API が正しく動いているか確認できる

DevTools を使えるようになると、**開発の効率が何倍にもなります！**

---

## 学習の進め方

## クイックスタート

### 今すぐ試してみよう！

1. **DevTools を開く**
   - Windows / Linux: `Ctrl + Shift + I`
   - Mac: `Command + Option + I`

2. **要素を検証してみる**
   - 何かボタンを右クリック → **「検証」**
   - Elements パネルでコードを確認

3. **Console でエラーを確認**
   - Console タブを開く
   - 赤色のエラーがあるか確認

4. **CSS を変更してみる**
   - Elements パネルで要素を選択
   - 右側の Styles パネルで色やサイズを変更

**これだけで DevTools の基本がわかります！**

## よくある質問

### Q1. DevTools で変更した内容は保存されますか？

**A:** いいえ、DevTools での変更は **一時的** です。ページをリロードすると元に戻ります。

気に入った変更があれば、コードをコピーして実際のファイルに反映してください。

---

### Q2. どのブラウザで使えますか？

**A:** この教材は **Google Chrome** 向けですが、以下のブラウザでも同様の機能が使えます：

- Microsoft Edge（Chromium版）
- Brave
- Vivaldi
- その他 Chromium ベースのブラウザ

Firefox や Safari にも開発者ツールがありますが、UIや機能が少し異なります。

---

### Q3. プログラミング初心者でも理解できますか？

**A:** はい！この教材は **非エンジニア向け** に作られています。

- 専門用語は必要最小限に抑えています
- 図解や具体例を多く使っています
- 実際に手を動かしながら学べる構成になっています

HTML / CSS / JavaScript の基礎知識があれば、より理解しやすいですが、なくても大丈夫です。

---

### Q4. どのくらいの時間で習得できますか？

**A:** 以下を目安にしてください：

- **基礎だけ学ぶ**：1〜2時間
- **主要パネルを使えるようになる**：3〜4時間
- **実践的に使いこなす**：5〜10時間

ただし、**実際に自分のプロジェクトで使う** ことが一番の近道です！

## 学習のコツ

### 1. 実際に手を動かす

読むだけでなく、必ず **自分で DevTools を開いて試してください**。

- 自分のプロジェクトで試す
- お気に入りのサイトで試す
- 練習用のサンプルページで試す

### 2. エラーを恐れない

DevTools で何を変更しても、**ページをリロードすれば元に戻ります**。

どんどん実験して、試してみましょう！

### 3. わからないことは検索する

エラーメッセージをコピーして Google で検索すると、解決策が見つかることが多いです。

- Stack Overflow
- Qiita
- MDN Web Docs

などが参考になります。

### 4. ブックマークする

よく使うページは **ブックマーク** して、すぐに見返せるようにしましょう。
